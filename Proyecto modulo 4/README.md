Proyecto Módulo 4– Aplicación de Consola en JavaScrip
 Descripción
Este proyecto corresponde al Módulo 4 del curso y consiste en una aplicación desarrollada en JavaScript que se ejecuta en la consola del navegador.

La aplicación permite al usuario ingresar su nombre, elegir una operación matemática y visualizar los resultados directamente en la consola.



 Tecnologías utilizadas
- HTML5
- JavaScript
- Navegador Google Chrome
- Visual Studio Code



 Funcionalidades
- Solicitud de datos al usuario mediante prompt
- Uso de variables y operadores matemáticos
- Implementación de funciones
- Uso de condicionales (if y switch)
- Manejo de arreglos y objetos
- Recorrido de arreglos con forEach
- Visualización de resultados en la consola




 Instrucciones de uso
1. Abrir el archivo index.html en un navegador web
2. Ingresar el nombre del usuario cuando se solicite
3. Ingresar los números requeridos
4. Seleccionar la operación deseada
5. Revisar los resultados en la consola del navegador
